World Wide Web:

The World Wide Web got such a sucess because of essential technologies. The most essential technologies are Mosaic, Navigator and the Internet Explorer.
With a URL you can address the ressources.
A URL includes: a protocol, a hostname and a filename. Aswell it includes scheme, host authority(userinfo, host, port), path, query and fragment.

Web Server:

A web server is computer software and underlying hardware that accepts requests via HTTP or its secure variant HTTPS.
He intitiates communication by making a request for a web page or other resource using http and responds to that.
With a file system content can be stored on the web server.
The content is described with the usage of css.

Web Client:

A web client is nothing else than a web browser.
A web client fetches content from the web and displays it.

html:

The beginning of an html file starts with !Doctype html and its information to the browser of what document to expect.
head and body are the most used elements of an html file.

css


The components of a css rule are a selector, a value, a property and a declaration.
Descendant Selector: using space ex.: div p Child Selector: div > p
Adjacent Sibling Selector: div + p : The adjacent sibling selector is used to select an element that is directly after another specific element.
General Sibling Selector: div ~ p : The general sibling selector selects all elements that are next siblings of a specified element.

Commands

1. Declarations color/bgcolor: style="background-color: Tomato" style="color: Tomato"
2. background image: background-image: url(xy.jpg);
3. text alignment: text-alignment: center;
